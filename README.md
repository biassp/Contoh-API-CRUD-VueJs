#VueJS Contoh CRUD Application
Aplikasi ini adalah contoh pembuatan halaman CRUD menggunakan VueJS dan mengakses data melalui API.

# Install

    > npm install

API yang digunakan dalam contoh ini adalah Node.js express API.

# Run

    > npm run dev

Navigasi ke http://localhost:8080 untuk melihat situs

# Start API

Untuk memulai API, buka terminal perintah Anda ke lokasi ini dan masuk

    > node server.js

API sekarang aktif di http://localhost:3000
